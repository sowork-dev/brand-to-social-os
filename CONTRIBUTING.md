# Contributing

Thanks for contributing.

## Good contributions

We are especially interested in:

- clearer positioning frameworks
- stronger examples
- better social operating checklists
- channel-specific add-ons for LinkedIn, X, Instagram, TikTok, and newsletters
- installation and portability improvements

## Rules

- Do not add client names, private datasets, tokens, cookies, or internal URLs.
- Keep examples sanitized and generic.
- Prefer practical templates over abstract theory.
- When adding claims about market demand, include a source link or clear date.

## Pull request checklist

- [ ] no secrets or credentials
- [ ] no customer-sensitive information
- [ ] examples are generic or fictionalized
- [ ] docs updated if behavior changed
