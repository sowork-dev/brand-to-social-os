# Content Pillar Design

Use this skill to make content production easier and more coherent.

## Rule
Each pillar should connect one strategic idea to many repeatable post formats.

## For each pillar define
- strategic purpose
- audience question it answers
- formats it supports
- proof source
- CTA tendency

## Recommended pillar set

### 1. Education
Teach the market how to see the problem more clearly.

### 2. Point of view
Show your strategic opinion and category contrast.

### 3. Proof
Show work, process, examples, outcomes, or before/after evidence.

### 4. Behind the scenes
Humanize the operator and expose decision quality.

### 5. Conversion
Invite the next step without making the whole feed promotional.

## Output format

```markdown
# Content Pillars

## Pillar 1
- purpose
- audience question
- repeatable formats
- proof source
- CTA

## Pillar 2
...
```
