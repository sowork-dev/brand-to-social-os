# LinkedIn Operator Pack - Pre-publish Checklist

Run this before publishing any LinkedIn post.

## Post quality check

- [ ] The hook line is one sentence and names a specific tension, fact, or counterintuitive claim
- [ ] The hook does not start with "I" or "We" or "Excited to announce"
- [ ] The body uses short paragraphs with generous line breaks
- [ ] The post is written in first person throughout
- [ ] No links are in the post body (they go in the first comment)
- [ ] The close is a question, direct statement, or quiet CTA, not a list of hashtags
- [ ] If using hashtags: max 3, only at the very end

## Strategic check

- [ ] This post serves one content pillar clearly
- [ ] The post connects back to the brand positioning in some way
- [ ] The CTA tendency matches the pillar (education posts rarely ask for demos)

## Red flags

- [ ] Hook is a generic observation anyone could have written. Rewrite with specificity.
- [ ] Body explains before it hooks. Move the tension to line 1.
- [ ] Post sounds like a company announcement. Rewrite in first person.

## After publishing

- [ ] Reply to every comment within 60 minutes
- [ ] Note which comments show strong resonance or objection
- [ ] Add any strong objections to the experiment backlog
- [ ] DM anyone who asks a specific question worth continuing
