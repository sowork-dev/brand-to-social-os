# Sanitization Policy

This repository was prepared for public release.

## Removed on purpose

The public version excludes:

- customer names and private workspaces
- internal database names and schemas
- live URLs that expose private environments
- screenshots from customer systems
- prompt fragments tied to private data
- operational notes containing credentials or account structure
- unpublished campaign performance data

## Allowed in public examples

Public examples should be:

- fictionalized
- aggregated
- generic enough to teach the workflow
- useful without leaking commercial context

## Rule of thumb

If a file would make a client uncomfortable when posted to GitHub, it does not belong here.
